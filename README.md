# IteaLesson4HomeWork

1) Создать калькулятор, который принимает число, базовый математический оператор (+,-,*,/,^) и второе число из пользовательского ввода, и распечатывает результат математической операции. Математические операции должны быть заключены в функции.

2) Написать функцию, которая печатаем Пирамиду на консоль. Используя пользовательский ввод, задать высоту пирамиды и символ, которым будет строится пирамида. Пример вывода:

```
	               **
	              ****
	             ******
	            ********
	           **********
	          ************
```

3) Написать функцию, которая позволит вычитать факториал числа. Сделать 2-е реализации ( через рекурсию и через циклы ).

4) Написать функцию для поиска простых чисел в заданном диапазоне.

Минимальное число заданий, которые необходимо выполнить - 2.
